## Overview
This tool is a PowerShell diagnostic utility designed to quickly validate network connectivity, proxy configuration, port reachability, ping response, and certificate health from a Windows machine.

It is intended for system administrators and support engineers who need a fast way to troubleshoot connectivity and certificate-related issues in enterprise environments.

> ⚠️ **Important:** This script must be executed from an **elevated PowerShell ISE** (Run as Administrator).

### Features
The script provides interactive options to test:
* Proxy configuration and reachability
* Network port availability
* ICMP Ping tests
* SSL/TLS certificate validation
* Certificate chain and expiration diagnostics
